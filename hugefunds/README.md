## 🎯 Usage Examples

### Calculate Risk
```bash
curl -X POST http://localhost:8000/api/risk/cvar \
  -H "Content-Type: application/json" \
  -d '{"returns": [0.01, -0.02, 0.015, -0.01, 0.02], "confidence": 0.95}'
```

### Run Stress Test
```bash
curl -X POST http://localhost:8000/api/risk/stress-test \
  -H "Content-Type: application/json" \
  -d '{"positions": [{"symbol": "AAPL", "quantity": 100, "current_price": 175, "side": "long"}]}'
```

### Pre-Trade Check
```bash
curl -X POST http://localhost:8000/api/governance/pre-trade-check \
  -H "Content-Type: application/json" \
  -d '{"signal": {"confidence": 0.75}, "portfolio": {"var_95": 0.02}}'
```
